IoT Hands On: Build a Realtime App with Raspberry Pi 2 and PubNub
Class - Location TBD
Walk out of this workshop with your own IoT application, and a Raspberry Pi 2! Join Jasdeep, Tomomi and Bhavana as they walk you through building a realtime Internet of Things application using sensor data from Raspberry Pi 2. Just released, this hardware makes it quick and easy to build and deploy internet-enabled sensors. You'll also learn how to aggregate that sensor data and display it with realtime updates over the PubNub network. Apache Spark Workshop (w/ Sujee Manian)
Workshop - Location TBD
Apache Spark is emerging as next generation of technology for data processing. Compared to Hadoop MapReduce, Spark is easier to use and maintain, and Spark’s stream processing capability makes it a very good fit for processing IoT data.

This workshop will introduce Spark to attendees. We will go through the Spark concepts and learn how to use Spark through on hands-on labs solving practical, real world problems.

Attendees *must* have a working Apache Spark environment pre-installed on their laptop. [Download Spark]
